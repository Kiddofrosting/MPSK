DEPLOYMENT
----------
The site is ready to go live. We have configured it for one-click deployment on
Render.com. The steps are straightforward:

  1. A MongoDB Atlas account is created (free) — this is where all school data lives.
  2. The code is pushed to GitHub.
  3. A Render Web Service is created and connected to the repository.
  4. Environment variables (database URL and secret key) are set in Render dashboard.
  5. The site is live within minutes at an HTTPS URL (e.g. mpsk-kabarak.onrender.com).
  6. A custom domain (e.g. mpsk.ac.ke) can be pointed to Render at no extra charge.
